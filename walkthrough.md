# 작업 완료 보고: ST Lab 고도화

## 1. 개요
ST Lab의 분석 기능 강화와 사용성 증대를 위해 **실시간 센서 모니터링 그래프(Live Sensor Monitor)** 기능과 **가상 장치 프로필 선택** 기능을 구현했습니다.

## 2. 주요 구현 사항

### 2.1 실시간 센서 모니터링 (Live Monitor Ttap)
*   **시각화**: `recharts`를 활용하여 SSE 이벤트로 수신되는 센서 데이터(온도, 습도, 전력량, 배터리 등)를 실시간 차트로 시각화했습니다.
*   **자동 감지**: 별도의 설정 없이, 수신된 데이터 중 수치형(Number) 속성을 자동으로 감지하여 차트에 추가합니다.
*   **다중 속성 지원**: 여러 속성을 동시에 그래프에 표시할 수 있으며, 상단의 토글 버튼으로 특정 속성만 필터링하여 볼 수 있습니다.
*   **실시간성**: SSE 스트림과 직접 연동되어, 데이터 변경 즉시 차트가 업데이트됩니다.

### 2.2 가상 장치 생성 고도화
*   **프로필 선택**: 기존에 이름만 입력하던 방식에서 벗어나, **다양한 기기 프로필(스위치, 디머, 센서 등)**을 선택하여 생성할 수 있도록 개선했습니다.
*   **UI 개선**: 프로필 목록을 직관적인 그리드 형태로 제공하여 선택의 편의성을 높였습니다.

## 3. 사용 가이드
1.  **Monitor**: 디바이스 선택 > `Monitor` 탭으로 이동. (SSE 연결이 필요합니다.)
2.  **Virtual Device**: `Tools` 탭 > `Virtual Device Manager` > `Device Type`에서 원하는 센서 종류 선택 후 생성.

## 4. 향후 계획
*   차트 데이터 로깅 및 CSV 내보내기 기능 추가 고려.
*   가상 장치의 더 다양한 프로필 추가.
