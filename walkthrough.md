# Tizen Lab 앱 목록 파싱 오류 수정 Walkthrough

## 변경 사항
Tizen 기기에서 앱 목록을 가져올 때 발생하는 파싱 오류를 수정했습니다.

### `server/index.cjs`
- `list_tizen_apps` 핸들러의 파싱 로직을 개선했습니다.
- **정규식 업데이트**: 상태 표시가 대괄호 `[]`로 묶인 경우도 처리할 수 있도록 정규식을 `/^(\S+)\s+(.+?)\s+[(\[](.+)[)\]]$/`로 변경했습니다.
- **헤더 필터링**: `system vpkg_type [WGT]`와 같은 헤더 라인이 앱 카드로 표시되지 않도록 필터링 로직을 추가했습니다.

## 확인 방법
1. 서버(Node.js 프로세스)를 재시작합니다.
2. HappyTool의 Tizen Lab > App Manager 탭으로 이동합니다.
3. 앱 목록을 새로고침하여 실제 설치된 앱들이 정상적으로 표시되는지 확인합니다.
