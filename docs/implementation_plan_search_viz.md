# 검색 결과 시각화 방식 변경 (하이라이트 대신 딤드) 🎨

검색 결과(태그 및 플러그인 태그)가 있을 때, 검색된 항목을 파란색으로 하이라이트 하는 대신 **검색되지 않은 항목을 어둡게(Dimmed) 처리**하여 검색된 항목이 도드라져 보이게 개선합니다.

## 제안 사항 🐧

- 검색어가 있을 때 검색된 항목은 원래 색상(또는 Fail 색상)을 유지하고 전체 불투명도로 표시합니다.
- 검색되지 않은 항목은 `Fail Only` 모드와 유사하게 매우 낮은 불능도(0.08)로 처리합니다.
- 검색 결과 하이라이트 시 사용하던 파란색(`6366f1`) 배경색 적용 로직을 제거합니다.

## 변경 예정 파일 🛠️

### [MODIFY] PerfFlameGraphRenderer.ts
- `drawFlameChart` 함수 내의 색상 및 투명도 결정 로직 수정.
- `isMatch` 시 파란색으로 바꾸던 로직 제거.
- 검색 중일 때 `!isMatch`인 항목에 대해 `baseOpacity` 축소 적용.

## 검증 계획 ✅

- 검색어 입력 시 검색된 항목이 원래 색상으로 밝게 유지되는지 확인.
- 검색되지 않은 항목이 어둡게 변하는지 확인.
- `Fail Only` 모드와 함께 사용 시에도 정합성 있게 동작하는지 확인.
