# nupkg signer 타임아웃 문제 해결 계획서

## 1. 개요
`nupkg-auto-sign-so` IPC 핸들러에서 발생하는 '자동 서명 대기 시간 초과(20초)' 에러를 해결하기 위해 로직을 개선하고 디버깅 정보를 강화합니다.

## 2. 변경 사항

### 2.1. electron/main.cjs 수정
- **타임아웃 시간 연장**: `waitTime`을 20000(20초)에서 60000(60초)으로 변경.
- **단계별 로깅 추가**:
    - ISMS 페이지 로드 완료 시점.
    - `year`, `fileType`, `file_upload_1` 요소 탐색 성공 시점.
    - CDP를 통한 파일 업로드 시도 및 완료 시점.
    - `#download_link` 폴링 시작 및 발견 시점.
    - 다운로드 시작(`will-download`) 및 완료 시점.
- **에러 메시지 상세화**: 타임아웃 시 현재 브라우저의 상태(URL, Title)뿐만 아니라 마지막 성공 단계를 기록하여 출력.

### 2.2. 디버깅 강화
- 요소 탐색 실패 시 현재 프레임 구조를 콘솔에 출력하도록 수정.

## 3. 기대 효과
- 서버 지연에 따른 불필요한 타임아웃 방지.
- 문제 발생 시 정확히 어느 지점에서 멈췄는지 파악 가능하여 향후 유지보수 용이.
