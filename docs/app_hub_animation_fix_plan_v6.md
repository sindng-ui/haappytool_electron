# Implementation Plan - App Hub "Randomized Entrance & Bouncy Hover"

형님, 역시 그 제각각 노는 리듬감이 형님의 취향이셨군요! 🐧✨
제가 너무 부드럽게 한 세트로 움직이게 만든 게 실수였습니다. 각 카드마다 고유의 성격(ID 길이 기반 파라미터)을 다시 부여해서, 형님이 좋아하시던 그 **'뚝뚝' 끊기듯 제각각 튀어나오는 리듬감**을 완벽하게 되살리겠습니다.

## 📋 변경 사항

### 1. `components/AppLibraryModal.tsx`

- **등장 애니메이션 (Entrance) - 개별 리듬 복구**
  - `visible` 변이의 `transition` 파라미터를 다시 **개별 ID 기반**으로 분산시킵니다.
  - `stiffness`, `damping`, `mass`, `delay`를 모두 각각 다르게 설정하여, 어떤 놈은 빨리, 어떤 놈은 천천히, 어떤 놈은 묵직하게 나오도록 만듭니다.
  - 특히 `delay` 편차를 더 크게 주어 '다들 기다렸다가 제각각 움직이는' 느낌을 극대화하겠습니다.
- **마우스 오버 (Hover) - 뾰뵹함 유지**
  - 마우스 오버 시의 쫀득한 탄성(`bounce: 0.4`)은 그대로 유지합니다.

## 🛠️ 작업 단계

1. `AppCard`의 `visible` transition에 개별 랜덤 로직 재도입.
2. 등장 시의 무작위성(Randomness) 수치 강화.
