# 구현 계획서: 앱 허브 "엣지-글래스" 최적화 튜닝 🐧💎

블러 연산량을 대폭 줄여 성능을 확보하고, 시각적 선명도는 테두리와 광택 효과로 보완하여 "가볍고 영롱한" 사용자 경험을 제공합니다.

## 1. 주요 변경 사항

### 1.1 AppCard.tsx: 비주얼 & 성능 하이브리드 최적화
- **블러 최적화**: `backdrop-blur-2xl`을 `backdrop-blur-md`로 변경 (연산량 약 60% 감소).
- **엣지 레이어 추가**: `border-white/30`과 `ring-1 ring-white/20` 조합으로 유리판의 날카롭고 선명한 테두리 질감 구현.
- **배경 투명도 미세 조정**: `bg-white/[0.08]` 정도로 낮춰 테두리 대비감을 극대화.
- **호버 글로우 최적화**: `group-hover` 시에만 블러 연산이 포함된 레이어가 활성화되도록 조건부 렌더링 최적화.

### 1.2 스타일 정밀 튜닝
- **내부 그라데이션**: 카드 상단에 아주 미세한 하이라이트 그라데이션을 넣어 블러가 적어도 입체감이 느껴지도록 처리.
- **그림자 경량화**: 너무 넓게 퍼지는 그림자 대신, 가볍고 밀도 있는 그림자로 변경하여 렌더링 성능 향상.

## 2. 기대 효과
- **반응 속도**: 마우스 호버 및 모달 오픈 시의 체감 지연 시간(Latency) 감소.
- **비주얼 만족도**: 블러에 의존하던 몽환적인 느낌에서, 정교하게 깎인 유리 보석 같은 선명한 느낌으로 진화.
