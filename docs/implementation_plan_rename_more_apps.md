# Implementation Plan - App Hub UI 문구 변경 🚀

형님! App Hub의 'More Apps' 문구를 'Labs'로 변경하여 더 세련되고 전문적인 느낌을 주도록 하겠습니다. 🐧✨

## 1. 개요
현재 App Library 모달에서 고정되지 않은 앱들을 보여주는 섹션의 타이틀이 "More Apps"로 되어 있습니다. 이를 유저의 요청에 따라 "Labs"(실험실)로 변경합니다.

## 2. 변경 사항

### [[App Library UI 수정]]
- **대상 파일**: `components/AppLibraryModal.tsx`
- **수정 내용**: 
  - `Section` 컴포넌트의 `title` 프롭 값을 `"More Apps"`에서 `"Labs"`로 변경.

### [[APP_MAP 업데이트]]
- **대상 파일**: `important/APP_MAP.md`
- **수정 내용**: 
  - Section 1.3의 키워드 혹은 설명에 "Labs" 개념 반영

## 3. 작업 절차
1. `components/AppLibraryModal.tsx` 파일 수정
2. `important/APP_MAP.md` 파일 업데이트 (UI 변경 사항 반영)
3. 변경 사항 확인
